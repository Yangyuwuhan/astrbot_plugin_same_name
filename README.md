## 简介

一个为 AstrBot 设计的插件，用于查询全国重名人数和相关信息

## 安装

克隆本项目到 `/data/plugins` 插件目录中，重启 AstrBot 

## 使用

- 首次使用时，需要注册 [枫林 API](https://www.yuafengapi.com/) 免费获取密钥
- 使用指令 `/重名 [姓名]` 进行查询
